# Features

- 물리 시뮬레이션
- 인터렉티브 그래픽
  </br>

# Prerequisites

- Node.js 18.x or higher

# Installation

```bash
git clone https://github.com/enkoding1/creative-coding
cd creative-coding
npm install
npm run dev
```
